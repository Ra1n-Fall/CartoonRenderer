## 알고리즘의 한계점 

이미지를 블러처리하여 엣지를 검출하고 해당 엣지를 원본 이미지에 마스킹하는 방식이다. 따라서 경계선이 뚜렷하지 않거나 너무 두꺼운 경우 엣지 검출에 있어 결과가 바람직하지 못하게 나올 수 있다.
